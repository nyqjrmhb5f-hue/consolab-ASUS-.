# 02_COMMERCIAL_ROOM

Primary engine: `REVENUE-NEXUS`

This room owns order-to-cash, pricing, contracts, accounts, subscriptions, revenue, and forecasting. `PROMO-ARC` lives here as the promotion and incentive engine.

Key flows:
- `pricing/` and `contracts/` shape commercial intent
- `opensell/` and `promotions/` drive acquisition and conversion
- `evidence_refs/` links signed commercial actions into `04_EVIDENCE_ROOM`
